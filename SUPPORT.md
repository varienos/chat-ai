# Support

## Getting Help

For general setup, deployment, or usage questions:

- Start with [README.md](README.md).
- Check [SECURITY.md](SECURITY.md) before reporting security-sensitive issues.
- Open a GitHub issue for reproducible bugs or documentation gaps.

## Live Example

- Widget example: https://varien.software
- API example: https://varien.software/varien-ai

The public Varien production endpoint is not a shared hosted service for this
repository. Deploy your own gateway and configure the widget to point at your
own hostname.
