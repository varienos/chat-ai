## Suggested structure

Group by domain so the assistant can answer "do you do X" questions cleanly.

## Example

### Frontend

- **Web:** React, Next.js, Vue, Svelte, Tailwind CSS
- **Mobile:** React Native, Flutter, native iOS (Swift), native Android (Kotlin)

### Backend

- **Languages:** TypeScript / Node.js, Python, Go, Ruby
- **Frameworks:** Fastify, Express, NestJS, FastAPI, Django, Rails
- **Realtime:** WebSockets, Server-Sent Events, WebRTC

### Data & Infra

- **Databases:** PostgreSQL, MySQL, MongoDB, Redis, Elasticsearch
- **Cloud:** AWS, GCP, Azure, Cloudflare, Vercel, self-hosted (Docker, k8s)
- **CI/CD:** GitHub Actions, GitLab CI, ArgoCD

### AI / ML

- **Foundation models:** OpenAI, Anthropic Claude, Google Gemini, open-source LLMs
- **Tooling:** LangChain, LlamaIndex, vector DBs (Pinecone, Qdrant, pgvector)
