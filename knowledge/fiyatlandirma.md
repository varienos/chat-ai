# Fiyatlandırma ve Paketler

## Mobil Uygulama Geliştirme

### Başlangıç / MVP Paketi — 70.000 TL - 120.000 TL

- Tek platform veya cross-platform (iOS + Android)
- 10-15 ekran
- Temel UI/UX tasarım
- Basit backend API + admin paneli
- 3 ay ücretsiz destek
- Tahmini süre: 6-12 hafta
- Örnek projeler: eğitim uygulaması, basit e-ticaret, randevu sistemi

### Profesyonel Paket — 150.000 TL - 250.000 TL

- iOS + Android (cross-platform veya native)
- 15-30 ekran
- Özel UI/UX tasarım
- Kapsamlı backend, admin paneli, ödeme entegrasyonu
- Push bildirimler, analitik entegrasyonu
- 6 ay ücretsiz destek
- Tahmini süre: 10-16 hafta
- Örnek projeler: abonelik platformu, konum bazlı eşleştirme, AI destekli uygulama

### Kurumsal Paket — 250.000 TL+

- Native iOS + Android veya kapsamlı cross-platform
- 25+ ekran, karmaşık iş mantığı
- Özel tasarım sistemi
- Mikroservis mimarisi backend
- 3. parti entegrasyonlar (ödeme, harita, ERP, HBYS)
- AI/ML özellikleri
- 12 ay destek + SLA
- Tahmini süre: 16-40 hafta
- Örnek projeler: hastane uygulaması, IoT platformu, marketplace

### Büyük Ölçekli Platform — 350.000 TL - 500.000 TL+

- Çoklu kullanıcı rolleri (admin, işveren, kullanıcı, öğretmen vb.)
- E-ticaret, abonelik, cüzdan/ödeme sistemi
- Gamification (puan, seviye, ödül sistemi)
- Birden fazla modül (eğitim, etkinlik, mesajlaşma)
- Fazlı geliştirme önerilir (MVP + genişleme fazları)
