# Services / Hizmetler

> **Knowledge base template** — Replace this content with the services or
> products you want the assistant to discuss. This file is injected into
> every chat prompt, so write it in the voice and language you want responses
> to use.

## Suggested structure

For each service, document:

- **Name** — short, recognizable
- **Description** — 1–2 sentences about scope and outcome
- **What's included** — concrete deliverables
- **Typical timeline** — weeks or sprints
- **Starting point** — "from $X" or "contact for a quote"
- **Good fit for** — who should use this service

## Example

### Web Application Development

End-to-end web platforms with custom backend, admin tooling, and integrations.

- **Includes:** discovery → UX → frontend + backend → deploy → 30-day support
- **Timeline:** 8–14 weeks
- **Starting from:** contact for a quote
- **Good fit for:** SaaS MVPs, internal tools, customer portals
