# Kullandığımız Teknolojiler

## Frontend & UI

- **React.js** — Ana frontend kütüphanemiz
- **Next.js** — SSR ve SSG destekli React framework
- **Tailwind CSS** — Utility-first CSS framework
- **Framer Motion** — Animasyon kütüphanesi
- **Vue.js** — Alternatif frontend çözümleri için
- **Redux / Zustand** — State management
- **Three.js** — 3D görselleştirme
- **Storybook** — Component dokümantasyonu
- **Radix UI** — Erişilebilir UI componentleri

## Mobile

- **Swift / SwiftUI** — iOS native geliştirme
- **Kotlin** — Android native geliştirme
- **Flutter** — Cross-platform (Dart ile)
- **React Native** — Cross-platform (JavaScript ile)
- Platform kapsamı: iOS %98, Android %95

## Backend

- **Node.js / Express / NestJS** — JavaScript/TypeScript backend
- **Python** — AI/ML projeleri ve otomasyon
- **Go** — Yüksek performanslı mikroservisler
- **PostgreSQL** — Ana ilişkisel veritabanı
- **MongoDB** — NoSQL doküman veritabanı
- **Redis** — Cache ve session yönetimi
- **GraphQL** — API sorgulama dili
- **Prisma** — ORM
- **Supabase / Firebase** — BaaS çözümleri

## DevOps & Cloud

- **AWS** — EC2, S3, Lambda, Amplify
- **Docker & Kubernetes** — Konteynerizasyon ve orkestrasyon
- **CI/CD Pipeline** — Otomatik derleme, test ve dağıtım
